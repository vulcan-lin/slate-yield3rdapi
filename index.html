POST /auth

Auth with Yield Dance. You must OAuth with mixin firstly to obtain the mixin token

Body parameter

{
  "mixin_token": "string"
}

Parameters

Name In Type Required Description
body body AuthReq true Yield Dance Auth request

Example responses

  "token": "string"
}

Responses

Status Meaning Description Schema

400 Bad Request Common response commonResponse
